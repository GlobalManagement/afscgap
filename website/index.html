AFSC GAP for Python

Community contributed Python-based tools for working with public bottom trawl surveys data from the NOAA Alaska Fisheries Science Center Groundfish Assessment Program (NOAA AFSC GAP). In addition to visual analytics tools that do not require code, it provides Pythonic access to the official API service with automated pagination, convienent contextual documentation, memory efficient zero catch (absence data) inference, and easy query generation / emulation for data filtering.

Download static datasets and other artifacts

To support research and other innovative work, this project also makes the following available under the CC-BY-4.0:

  • CSV file containing metadata of all AFSC GAP hauls: Download CSV hauls data. This is a frequently used static file retrieved from the UI at official NOAA AFSC GAP download website.
  • SQLite database with all species weights, counts, and catch effort with zero catch inference aggregated by 4 character geohash: Download SQLite geohash data. This is generated by build_database.py.
