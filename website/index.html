AFSC GAP for Python

Community contributed Python-based tools for working with public bottom trawl surveys data from the NOAA Alaska Fisheries Science Center Groundfish Assessment Program (NOAA AFSC GAP). It provides Pythonic access to the official API service with automated pagination, convienent contextual documentation, memory efficient zero catch (absence data) inference, and easy query generation / emulation for data filtering.
